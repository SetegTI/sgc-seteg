# SGC SETEG - Sistema de Gestão Cartográfica

Sistema de gerenciamento de solicitações cartográficas desenvolvido para o Setor Geoambiental.

### GitHub Pages

1. O site está disponível em `https://setegti.github.io/sgc-seteg/`

## 🔧 Tecnologias

- HTML5
- CSS3
- JavaScript 
- Firebase Realtime Database
- Bootstrap Icons

## 📁 Estrutura do Projeto

```
sgc-seteg/
├── index.html              # Página principal
├── css/
│   └── style.css          # Estilos da aplicação
├── js/
│   ├── app.js             # Lógica principal
│   ├── firebase/
│   │   └── firebase-config.js
│   ├── modules/
│   │   ├── auth-module.js
│   │   ├── cartography-data-module.js
│   │   └── map-module.js
│   └── utils/
│       └── dom-utils.js
└── assets/
    ├── images/            # Imagens (logos, backgrounds)
    └── icons/             # Ícones
```
